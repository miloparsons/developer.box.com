# ヘッダー

ページでヘッダーを作成できます。ガイドではヘッダーを使用しないでください。

```html
<Header to='/guides'>
  Guides
</Header>
```

<H>

<Header to="/guides" stroke centered uppercase>
Guides

</Header>
<H>

<Message>

オプションとして、ヘッダーに`centered`、`stroke`、`right`揃え、サイズ`small`を指定できるほか、ヘッダーを`uppercase`に自動変換したり、`light`バリアントで表示したりすることもできます。

</Message>
