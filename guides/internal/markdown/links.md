# リンク

## 参照リンク

可能であれば`[text][link_name]`構文を使用して参照リンクを使用し、実際のリンクをページの一番下に移動するようにしてください。

```md
# Title

Some linked [text][link_name]. And some more content.

And here is some more text.

[link_name]: https;//box.com
```
