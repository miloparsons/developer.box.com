# コードブロック

<!-- markdownlint-disable code-fence-style -->

SDKまたはCLIのドキュメントにすべてのコードサンプルが含まれているわけではありません。新しいコードサンプルを追加するには、標準のマークダウンバッククォートで囲みます。

````sh
```js
console.log('Hello, World!')
```
````

<H>

```js
console.log('Hello, World!')
```

</H>

<Message>

適切な構文の強調表示が適用されるように、すべてのコードブロックに有効な言語を追加してください。

</Message>
