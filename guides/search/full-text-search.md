# 全文検索

指定した検索クエリに一致するファイルとフォルダをBoxアカウント内で検索するには、SDK内でクエリ検索メソッドを使用し、必要に応じてキーワードやフィルタを渡します。

使用可能な`fields`を使用して、検索用語と検索フィルタを定義します。次に、定義した条件を使用して検索エンドポイントを呼び出すと、一致するファイルとフォルダが返されます。
